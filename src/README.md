## API Documentations
### /user/change-profile
#### Required:
- oldUsername (will change to JWT later or other auth method if implemented)

#### Optional:
- newUsername
- newPassword
- newGender ('Perempuan', 'Laki-Laki', 'Lainnya', 'Roti Tawar')
- newBirthDate ('YYYY-MM-DD')
- newEmail 
- newPhoneNumber
